## Phase 4: Google Apps Script デプロイ

### ステップ4-1: Apps Scriptエディタを開く

- [ ] 作成したスプレッドシートを開く
- [ ] メニューの「拡張機能」→「Apps Script」をクリック
- [ ] 新しいタブでエディタが開く

### ステップ4-2: コードの貼り付け

- [ ] エディタに表示されている既存のコードをすべて削除
- [ ] `src/Code.gs` の内容をすべてコピー
- [ ] エディタに貼り付け

### ステップ4-3: CONFIG設定

エディタ内の`CONFIG`オブジェクトを編集：

```javascript
const CONFIG = {
  FOLDER_ID_INPUT: "ここに01_未処理フォルダのIDを貼り付け",
  FOLDER_ID_OCR_IN_PROGRESS: "ここに02_OCR中フォルダのIDを貼り付け",
  FOLDER_ID_PROCESSED: "ここに03_処理済みフォルダのIDを貼り付け",
  FOLDER_ID_REVIEW: "ここに04_要確認フォルダのIDを貼り付け",
  SPREADSHEET_ID: "ここにスプレッドシートのIDを貼り付け",
  SHEET_NAME: "シート1"
};
```

- [ ] `FOLDER_ID_INPUT` にステップ1-2で取得したIDを貼り付け
- [ ] `FOLDER_ID_OCR_IN_PROGRESS` にステップ1-2で取得したIDを貼り付け
- [ ] `FOLDER_ID_PROCESSED` にステップ1-2で取得したIDを貼り付け
- [ ] `FOLDER_ID_REVIEW` にステップ1-2で取得したIDを貼り付け
- [ ] `SPREADSHEET_ID` にステップ2-3で取得したIDを貼り付け
- [ ] `SHEET_NAME` が「シート1」であることを確認

### ステップ4-4: 保存

- [ ] エディタ上部の「保存」ボタン（フロッピーディスクアイコン）をクリック
- [ ] プロジェクト名を「名刺OCRシステム」に変更（任意）

## Phase 5: Gemini APIキーの安全な設定

### ステップ5-1: スクリプトプロパティを開く

- [ ] エディタ左側のメニューで「⚙️プロジェクトの設定」をクリック
- [ ] 下にスクロールして「スクリプト プロパティ」セクションを見つける

### ステップ5-2: APIキーを登録

- [ ] 「スクリプト プロパティを保存」をクリック

**確認**:
- [ ] プロパティ名が正確に `GEMINI_API_KEY` になっている
- [ ] 値にAPIキーが設定されている（●●●●で表示される）

## Phase 8: デプロイ後テスト

### テスト8-1: テスト画像の準備

- [ ] テスト用の名刺画像を1枚用意
  - スマートフォンで撮影、またはPCから画像をアップロード
  - JPEG形式推奨（HEIC形式は自動的に要確認フォルダへ移動されます）

### テスト8-2: 画像アップロード

- [ ] Google Driveで `01_未処理` フォルダを開く
- [ ] テスト画像をアップロード

### テスト8-3: 手動実行

- [ ] GASエディタに戻る
- [ ] 「▶実行」ボタンをクリック

### テスト8-4: 結果確認

**実行ログの確認**:
- [ ] 「処理開始: [ファイル名]」が表示される
- [ ] 「処理完了: [処理済] 会社名_氏名.jpg」が表示される
- [ ] 「計 1 件の名刺画像を処理しました。」が表示される
- [ ] エラーが表示されていない

**フォルダの確認**:
- [ ] `01_未処理` フォルダが空になっている
- [ ] `03_処理済み` フォルダにファイルが移動している
- [ ] ファイル名が「[処理済] 会社名_氏名.jpg」に変更されている

**スプレッドシートの確認**:
- [ ] 「シート1」に1行データが追加されている
- [ ] 12列すべてにデータが記録されている
- [ ] A列: 登録年月日が正しい
- [ ] B列: 氏名が抽出されている
- [ ] C列: 会社名が抽出されている
- [ ] I列: 「処理済み」と表示されている
- [ ] J列: 画像リンクをクリックすると元画像が開く

**操作ログシートの確認**:
- [ ] 「操作ログ」シートに実行履歴が記録されている
- [ ] ステータスが「SUCCESS」になっている
- [ ] ファイル数が「1」になっている

## Phase 9: トラブルシューティング

### エラーが発生した場合

**エラー**: 「指定されたシート名が見つかりません」
- [ ] スプレッドシートのタブ名が「シート1」であることを確認
- [ ] CONFIG.SHEET_NAME を正しいシート名に変更

**エラー**: 「APIキーが設定されていません」
- [ ] スクリプトプロパティで `GEMINI_API_KEY` が正しく設定されているか確認
- [ ] プロパティ名が正確に `GEMINI_API_KEY`（すべて大文字）であることを確認

**エラー**: 「Gemini API呼び出し失敗: 認証に失敗しました」
- [ ] Gemini APIキーが正しいか確認
- [ ] APIキーの有効期限が切れていないか確認
- [ ] Google AI Studioで新しいAPIキーを生成して再設定

**ファイルが `04_要確認` に移動される**
- [ ] HEIC形式の画像ではないか確認
- [ ] 画像が破損していないか確認
- [ ] 画像が鮮明であるか確認
